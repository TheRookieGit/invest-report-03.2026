§ 06   路径适用场景与风险提示

Path 1 适合你如果…

  • 对 AI 叙事有高度信念，不需要多元化分散
  • 3月31日前想对 CEG 有明确布局
  • 希望保留最多现金等待 TSLA 交付数据
  • 可接受 Beta 1.32，但不希望增加过多品种
  • 止损触发条件：GOOGL 跌破 $260 或 AI Capex 连续两季下滑

Path 2 适合你如果…

  • 认为 META 是 AI 应用层独立 Alpha 来源
  • 希望最大限度分散单股风险（HHI 最低）
  • 接受整体 Beta 略高但换取更好的叙事多元化
  • 不介意同时管理 6-7 个仓位
  • 止损触发：META 广告收入增速跌破 10%，或 TSLA 交付 < 350k

Path 3 适合你如果…

  • 希望有真正的对冲机制（TLT 负相关）
  • 对非系统性风险（单股暴雷）特别敏感
  • 不介意 5 年回报略低于 Path 1
  • 基本全仓部署，接受现金缓冲几乎归零
  • 止损触发：利率超预期上升导致 TLT 加速下跌，需重新评估对冲有效性
